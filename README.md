# 广东学法考试自动化脚本

## 学习地址
[点击这里进入学习地址](http://xfks-study.gdsf.gov.cn/)

## 项目简介
个人练手项目。  
因通过浏览器调用 `submitLearn` 进行快速学习，后台会有记录，于是制作一个自动学习脚本来减少风险。

## 环境要求
- **Python 3.10**
- **Selenium 库**
- **Firefox 浏览器**（115版驱动）

## 下载地址
[下载链接](https://www.123pan.com/s/ydsAjv-5Ge03)  
**注意：** 请关闭杀毒软件后再运行。

## 测试说明
因测试时间和条件有限，“学习所有课程”部分没有完全进行测试，但主要功能能正常使用，完成100%进度是没问题的，有BUG欢迎反馈。
